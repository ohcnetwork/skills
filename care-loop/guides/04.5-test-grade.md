# Step 4.5 — Grade the tests  (spawned as **`care-test-grader`** · Opus 4.8, frontmatter-bound · checker ≠ maker)

Invoke the reusable lens skill **`care-test-grade`** — it holds the premise, the three inputs, the
rubric, and the gate. This guide is only the loop wiring.

**Model self-check:** you must be Opus (judgment tier — [models.md](./models.md)). If you can tell
you're a smaller model, emit `BLOCKED: spawned on wrong model tier` to your agent log and stop.

- **Checker ≠ maker:** the grader is a **fresh subagent spawned without the Step-3 E2E author's
  transcript or reasoning** — it sees only the artifacts (the spec files, `criteria.md`,
  `intent.md`). Never hand it the maker's conversation or its justification for the specs.
- **Loop-invoked mode:** the run dir exists, so `care-test-grade` resolves its inputs from it —
  `criteria.md` (Step 1) and `intent.md` (`care-diff-review`, Step 4) — plus the Step-3 spec(s). No
  second intent reconstruction.

## Two modes: specs exist vs. no-spec

**When specs exist:** invoke `care-test-grade` as above — full rubric (`Wrong` / `Missing` / `Weak` /
faithfulness).

**No-spec mode** (trivial change with tests skipped, or the e2e track was skipped): **do not invoke
`care-test-grade`** — the skill requires specs to run. Instead, grade the **diff against
`criteria.md`** directly as an inline Opus checklist: for each criterion, read the diff and confirm
the implementation satisfies it. An **unmet criterion blocks** the push the same way `Wrong` does,
bounded by the convergence guard ([governance.md](./governance.md) §2). This ensures the criteria
gate is always enforced, even without specs.

## On the result

- **`Wrong`** (or unmet criterion in no-spec mode) blocks the push → loop back to the Step-3
  implementer/E2E author to fix, bounded by the convergence guard
  ([governance.md](./governance.md) §2). These loop-backs **count against the 5-round cap**.
- **`Missing` / `Weak` / faithfulness** are advisory → round summary / PR body, not auto-fixed.

Success exit adds: *no graded spec is `Wrong`; no criterion is unmet; uncovered criteria are listed
in the PR body / round summary.*
